페이지 소개

인터넷 프로그래밍 3번째 과제를 위해 만들어진 페이지입니다. 해당 페이지는 연세대학교 컴퓨터과학과 20학번 인터넷 프로그래밍 수업을 수강하고 있는 전재희에 의해 만들어졌습니다. 인터넷 프로그래밍 수업에서는 HTML, CSS, Javascript 등의 웹 언어를 다루고 있으며, 따라서 상품으로 각 언어들을 공부할 수 있는 책을 올려드렸습니다. 앞으로 client-side 뿐만 아니라 server-side까지도 다루게 되므로 상품을 더 추가할 예정입니다. 다들 열심히 공부하는 이번 학기 되시길 바랍니다.
